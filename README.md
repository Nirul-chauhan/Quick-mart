# Quick-Mart

A full-stack grocery shopping application.

## Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- PostgreSQL
- Alembic
- JWT Authentication

### Frontend
- React
- Vite
- Axios

## Features

- User Registration & Login
- Role-Based Authentication
- Countries, States, Cities
- Categories
- Stores
- Profile Management

## Run Backend

```bash
cd mart-backend
uvicorn app.main:app --reload
```

## Run Frontend

```bash
cd mart-frontend
npm install
npm run dev
```
